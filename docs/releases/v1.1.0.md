# Agentic Design System v1.1.0

The first repository-tagged ADS release turns the project into a repeatable distribution, not just
a useful source checkout.

- Installs ten skills for Claude Code, Codex, Cursor, OpenClaw, and Hermes.
- Bundles the runtime templates and six workflow runbooks the orchestrator references.
- Makes rendered browser evidence authoritative for accessibility, overflow, touch targets, states,
  screenshots, and judge completion.
- Adds a clean-project install matrix and one release gate for metadata and deterministic smokes.
- Documents exact installer IDs, destination paths, activation, updates, and no-CLI fallback.

See [`CHANGELOG.md`](../../CHANGELOG.md) for the complete initial-release inventory.
