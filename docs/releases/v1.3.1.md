# Agentic Design System v1.3.1

This patch closes the finding-local repair gap exposed by the v1.3.0 Pawprint benchmark.

- Adds an adjacent-action consistency sweep to both builder revision prompts and independent
  regrades.
- Requires status, instructional copy, visible emphasis, enabledness, and native control semantics
  to agree across primary, secondary, toolbar, and inline actions.
- Treats an enabled-looking contradiction in a read-only, disabled, offline, permission-limited,
  or destructive state as a major `cues_affordances` finding that prevents `satisfied`.
- Preserves active actions in unaffected states and requires fresh rendered evidence at every
  changed breakpoint.
- Extends the structured-findings smoke across the executable workflow, manual review runbook,
  UX state inventory, grader template, and installed orchestrator references.
- Closes the retained Pawprint regression at both breakpoints: eight rendered snapshots passed,
  the prior major finding closed, and the independent regrade returned `satisfied` with no new
  findings.
